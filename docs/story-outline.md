# หนึ่งคืนที่หอพัก — Story Outline

โทน: สยองไทยแบบค่อยๆ ขนลุก (slow-burn dread), ไม่ gore, ไม่มีการทำร้ายตัวเอง, ผี = บรรยากาศ+จิตวิทยา. เหมาะ 13+.
เล่นจบ 10-20 นาที/รอบ. ~40 nodes. จอเดียวแบบนิยาย (typewriter + choices).

## ตัวละคร/ฉาก

- **ผู้เล่น**: นักศึกษา/คนทำงานที่เพิ่งย้ายเข้าหอพัก "สินสมบูรณ์เพลส" คืนแรก มาถึงตอนพลบค่ำด้วยมอไซค์ตัวเอง
- **ป้าเฝ้าหอ (ป้าจำเนียร)**: นั่งเฝ้าโต๊ะหน้าหอมาหลายสิบปี พูดจาแปลกๆ ครึ่งเตือนครึ่งหลบเลี่ยง
- **เพื่อนบ้านห้อง 402 (พี่แนน)**: อยู่มานาน รู้เรื่องเก่าบางส่วน แต่ไม่อยากพูดตรงๆ
- **ห้อง 404**: ห้องล็อกตายที่ป้ายหน้าห้องบอก "ห้องว่าง" แต่ล็อกจากข้างนอกด้วยกุญแจอีกชุด
- ฉากประกอบ: ตู้น้ำหยอดเหรียญหน้าหอ, ราวตากผ้าดาดฟ้า, บอร์ดประกาศเก่าที่ล็อบบี้, ซอยเปลี่ยวข้างหอที่ได้ยินเสียงมอไซค์วิ่งผ่านทุกคืน

## Sanity (สติ 0-3, ไอคอนเทียน 3 เล่ม)

เริ่มที่ 3. Choice ที่เผชิญเรื่องน่ากลัวตรงๆ (ฟังเสียงเคาะต่อ, แงะห้อง 404, เจอเงาในกระจก, ได้ยินเสียงเรียกชื่อตัวเอง) ลดสติ -1. Choice ที่ตั้งสติ (โทรหาแม่, สวดมนต์, เปิดเพลง, ออกไปสูดอากาศ) เพิ่ม +1 (สูงสุด 3). สติ 0 → บังคับเข้า ending ตาย (`end_death`) ทันที ไม่ว่าจะเลือกอะไร.

## โครงเรื่องหลัก (Act structure)

### Act 1 — เย็นวันแรก (arrive_gate → arrive_room, ~10 nodes)
มาถึงหอ, เจอป้าเฝ้าหอ (คุยหรือไม่คุยก็ได้ แต่ป้าจะเรียกทันก่อนขึ้นห้องเสมอ), รับกุญแจ, สังเกตป้ายหน้าห้อง 404 ที่เขียน "ห้องว่าง" ทั้งที่ล็อกอยู่, เดินขึ้นไปห้อง 403 (ห้องตัวเอง ติดกับ 404), จัดของ, ได้ยินเสียงเคาะครั้งแรกจากผนัง.

Foreshadow clues วางไว้ตรงนี้: ป้าเฝ้าหอมองเหมือนไม่แน่ใจว่ามีคนมาจริงไหม, กระจกในห้องน้ำฝ้ามัวเช็ดเท่าไหร่ก็ไม่ชัด, โทรศัพท์โทรหาที่บ้านไม่ติดทั้งที่สัญญาณเต็ม.

### Act 2 — ดึกคืนแรก (investigate branch, ~20 nodes)
ผู้เล่นเลือกได้หลายทาง ไม่บังคับลำดับ: คุยกับพี่แนนห้อง 402 / ไปดูบอร์ดประกาศล็อบบี้ / ไปเปิดตู้น้ำหยอดเหรียญ (บรรยากาศ, jump-scare เบาๆ) / กลับมาฟัง-สำรวจห้อง 404. เก็บเบาะแส (ข่าวตัดที่บอร์ด, สมุดทะเบียนเก่าในลิ้นชักโต๊ะป้าตอนป้าเผลอ, คำบอกเล่าของพี่แนน) ที่ค่อยๆ ต่อเป็นเรื่องไฟไหม้ห้อง 404 ปี 2547 ที่มีผู้เช่าติดอยู่ข้างในเพราะประตูล็อกจากนอก. ระหว่างนี้แทรกจังหวะตั้งสติ (โทรหาแม่ได้ถ้ายังไม่เจอ no_one_hears_me, สวดมนต์) และเบาะแสลับ (ไม่มีเงาในกระจก, เคาะประตูเพื่อนบ้านแล้วไม่มีใครได้ยิน, เห็นชื่อตัวเองในทะเบียนเก่า).

### Act 3 — ไคลแมกซ์ก่อนเช้า (climax, ~8 nodes)
รวมเส้นทางกลับมาที่จุดตัดสินใจใหญ่หน้าห้อง 404 (หรือในห้องตัวเอง) ตอนตี 3. แยก 5 ปลาย:

1. **สติ 0 ระหว่างทาง** → `end_death` (ตาย) เสมอ ไม่ว่าจะเลือกอะไรในจุดนั้น
2. เลือก **"ทนไม่ไหว ออกจากหอคืนนี้เลย"** (มีให้เลือกได้หลายจุดตั้งแต่กลาง Act 2) → `end_flee` (หนีออกจากหอ)
3. เลือก **"ไม่คิดต่อแล้ว นอนต่อจนเช้า"** ที่ไคลแมกซ์ (ไม่ต้องมีเบาะแสครบ) → `end_survive_unaware` (อยู่รอดถึงเช้าแบบไม่รู้ความจริง)
4. เลือก **"เผชิญหน้ากับความจริง"** ที่ไคลแมกซ์ โดยมี `pieced_together_history` → `end_true` (รู้ความจริงทั้งหมด: เหตุไฟไหม้ปี 2547 ที่ห้อง 404)
   - **Deviation จากดราฟต์แรก**: แทนที่จะเช็ค "อย่างน้อย 2/3 เบาะแส" (ซึ่งต้องใช้ choices แบบ OR ซ้อนกันหลายชุดในเอนจิ้นที่รองรับแค่ requires แบบ AND) ลดรูปเหลือเงื่อนไขเดียวคือ `found_logbook == true` (เจอชื่อตัวเอง/ทะเบียนห้อง 404 จากโต๊ะป้า) ที่ node `pre_climax_gate` — เป็นเบาะแสที่หนักแน่นที่สุดเพียงพอจะเป็น gate ได้โดยไม่ต้องมี combinatorial choices เกินจำเป็น ข่าวตัด/พี่แนน ยังคงอยู่ในเกมเป็นเนื้อหาเสริมให้บรรยากาศและปูเรื่องแน่นขึ้น แต่ไม่ใช่เงื่อนไขบังคับ
5. ถ้าปลดล็อก secret gate (`no_reflection && no_one_hears_me && own_name_in_registry`) ตัวเลือกไคลแมกซ์จะมีตัวเลือกลับเพิ่ม **"มองเข้าไปในกระจกอีกครั้ง...จริงๆ"** → `end_ghost_twist` (เราคือผี — ผู้เล่นเองคือวิญญาณที่ตายจากอุบัติเหตุมอไซค์บนถนนก่อนถึงหอ ไม่ใช่คนที่ห้อง 404 ที่กำลังหลอก)

## Foreshadowing แผนละเอียด (ให้ honest, re-readable)

วางไว้ในเส้นทางหลักทุกเส้น (ไม่ผูกกับ flag) เพื่อให้ re-read แล้วเห็นเป็นปมได้:
1. เปิดเรื่อง: เสียงมอไซค์ไกลๆ + บรรยายว่าถนนหน้าหอมีจุดเสี่ยงโค้งอันตราย (ผู้เล่นเพิ่งขี่ผ่านมาเอง)
2. ป้าเฝ้าหอเซ็นสมุดรับกุญแจ มือสั่นเล็กน้อย พึมพำ "เอ๊ะ...ป้าจำไม่ได้ว่าห้องนี้ให้เช่าไปหรือยัง"
3. กระจกห้องน้ำฝ้ามัวผิดปกติ เช็ดเท่าไหร่ก็ไม่ชัดเงาตัวเอง
4. โทรศัพท์มีสัญญาณเต็มแต่โทรออกไม่ติด/ไม่มีใครรับสาย
5. เคาะประตูเพื่อนบ้านแล้วไม่มีเสียงตอบทั้งที่ได้ยินเสียงทีวีข้างใน
6. ตอนท้าย (ทุก ending ที่ไม่ใช่ twist) ยังคงทิ้งท้ายเป็นนัยเบาๆ ว่าเสียงมอไซค์ในซอยหยุดไปหลังเที่ยงคืน — เว้น ending twist ที่เฉลยว่าเสียงนั้นคือเสียงอุบัติเหตุของผู้เล่นเอง

## Ending map (สรุป)

| ending | endingId | เข้าถึงยังไง |
|---|---|---|
| ตาย | `death` | สติแตะ 0 เมื่อไหร่ก็ตาม (บังคับ) หรือเลือกแงะห้อง 404 คนเดียวตอนดึกโดยไม่มีเบาะแสและสติต่ำ |
| หนีออกจากหอ | `flee` | เลือก "ทนไม่ไหว ออกจากหอคืนนี้" ที่จุดใดก็ได้ใน Act 2-3 |
| อยู่รอดถึงเช้าไม่รู้ความจริง | `survive_unaware` | ที่ไคลแมกซ์เลือก "ไม่คิดต่อแล้ว นอนต่อ" โดยไม่ต้องมีเบาะแสครบ |
| รู้ความจริงทั้งหมด | `true_ending` | ที่ไคลแมกซ์เลือก "เผชิญหน้ากับความจริง" และมี `pieced_together_history` |
| เราคือผี (ลับ) | `ghost_twist` | ปลดล็อกด้วย `no_reflection && no_one_hears_me && own_name_in_registry` แล้วเลือกตัวเลือกลับที่ไคลแมกซ์ |

## Deviations from the initial draft (implemented in data/story.json)

- Dropped `has_flashlight` — the spec's flag list is illustrative ("เช่น"), and the flashlight didn't end up gating any content once the outline was fleshed out. Cutting it kept the flag list focused.
- `pieced_together_history` gate simplified to `found_logbook == true` only (see Act 3 note above) instead of a 2-of-3 threshold.
- Death is reached **only** via the sanity-0 forced-transition mechanic (per spec §1.3's literal wording), not via a separate explicit "risky choice" branch — this matches the engine's `deathBySanityNodeId` override and was verified reachable through actual aggressive play (multiple -1/-2 sanity choices stack past 0 before the climax).

## Node count target

35-50 nodes รวม 5 ending nodes. แผนจริง ~40-44 nodes เผื่อพื้นที่ให้บทสนทนาเดินเรื่องเป็นธรรมชาติ ไม่รวบรัดเกินไป.

# if-<slug> — Story Treatment Catalog (รอบ 1)

**Date:** 2026-07-04
**Purpose:** บทเยอะๆ หลายแนว ให้ บก. (peerapongsm) เลือก top picks → รอบ 2 ผมเขียน full branching script ให้ตัวที่เลือก แล้ว scaffold เป็นแอป `if-<slug>` แยกต่อแนว (reuse engine ของ if-horpak).

## กลไกที่ต้องแก้ใน engine (รอบ 2, ครั้งเดียวจบ)

if-horpak engine เกือบ generic แล้ว — node/choice/flags reuse ได้ทันที. จุดเดียวที่เป็นของแนวผีโดยเฉพาะคือ `sanity` (3→0 = `deathBySanityNodeId`). Generalize เป็น **named meter**:

```ts
// StoryData เพิ่ม (backward-compatible: horpak = { label:"สติ", max:3, floor:0 })
meter: {
  label: string;      // ชื่อโชว์ผู้เล่น เช่น "เวลา", "ใจเธอ", "เสบียง", "KPI"
  max: number;        // ค่าเริ่ม/เพดาน
  floor: number;      // ปกติ 0
  floorNodeId: string; // node ที่ถูกบังคับเข้าเมื่อ meter ถึง floor (เดิม = deathBySanityNodeId)
  hideBar?: boolean;   // บางแนวไม่โชว์บาร์ (เล่นแบบไม่รู้ตัว)
}
```

- `sanityDelta` บน choice → เปลี่ยนชื่อเป็น `meterDelta` (หรือ alias ทั้งคู่ช่วง migrate)
- บางแนวมี **2 meters** (เช่น survival: สุขภาพ + เสบียง) — ถ้าเลือกเรื่องแบบนั้น ค่อยขยายเป็น `meters: Record<string, Meter>` (ยังไม่ทำถ้าไม่จำเป็น — YAGNI)
- flags = ใช้ของเดิม (unlock true ending, จำการตัดสินใจ), endings meta = ของเดิม

แต่ละ treatment ด้านล่างระบุ meter ของตัวเอง.

## แนว A — สืบสวน / ระทึกขวัญ (mystery / thriller)

### A1 · «ห้องเช่าเลขที่ 8 หาย»
- **Logline:** คุณย้ายเข้าหอใหม่ พบว่าห้อง 8 ไม่มีอยู่ในผังชั้น ทั้งที่บันไดเดินขึ้นได้ถึง — คนที่เคยอยู่ห้องนั้นหายไปแบบไม่มีใครจำได้
- **Meter:** **เวลา** (คุณมี 3 คืนก่อนเจ้าของหอจะกลับมา; ยิ่งเสี่ยง/ผิดพลาด เวลายิ่งหด; ถึง 0 = ถูกจับได้/สายเกินไป)
- **Tone:** ระทึก เย็นๆ เมืองกรุง ไม่มีผี — ปริศนาคนหาย
- **Branch hook:** เชื่อผู้จัดการหอ vs สืบเอง; แอบเข้าห้องคนอื่น vs ตามเอกสาร
- **Endings:** จับตัวคนผิด (`true`) / รู้ความจริงแต่เงียบ (`silent`) / กลายเป็นคนต่อไปที่หาย (`vanish`) / ย้ายออกทัน (`flee`) / **บก.ตึกคือคุณในอนาคต** (`loop`, secret)
- **เปิดเรื่อง:** *ผังหนีไฟข้างลิฟต์เขียนไว้ครบทุกห้อง 1 ถึง 12 — เว้นเลข 8. คุณนับประตูตามระเบียงอีกรอบ หนึ่ง สอง สาม… ถึงบานที่ควรเป็นเลข 8 มันมีแค่รอยสี่เหลี่ยมซีดๆ ตรงที่เคยมีป้ายเลขห้อง. ประตูล็อกอยู่. ข้างในมีเสียงพัดลมหมุน.*

### A2 · «สายด่วน 1 นาที»
- **Logline:** คุณเป็นเจ้าหน้าที่รับสายฉุกเฉินกะดึก สายหนึ่งโทรเข้ามา เสียงกระซิบว่า "เขากำลังขึ้นบันไดมาแล้ว" แล้วสายตัด — คุณมีข้อมูลแค่เสียงพื้นหลังในการหาว่าเธออยู่ไหน
- **Meter:** **สัญญาณ** (ทุกครั้งที่ถามผิดคำถาม/เสียเวลา เธอยิ่งตื่นตระหนก สัญญาณ/สติเธอหด; ถึง 0 = สายขาดถาวร)
- **Tone:** เรียลไทม์ บีบหัวใจ ห้องเดียว หูฟังเดียว
- **Branch hook:** ปลอบให้เธอสงบ vs รีบซักที่อยู่; ส่งตำรวจไปจุดที่เดา vs รอข้อมูลเพิ่ม
- **Endings:** หาเจอทัน (`saved`) / ไปผิดที่ (`toolate`) / เธอรอด แต่คุณรู้ว่าคนร้ายคือใครที่คุณรู้จัก (`twist`) / สายหลอก (`prank`, secret)
- **เปิดเรื่อง:** *นาฬิกาบนจอขึ้น 02:47. หูฟังดังกรอบแกรบก่อนเสียงจะมา — ผู้หญิง หายใจถี่ พูดเบาจนคุณต้องเร่งวอลุ่ม. "อย่าเพิ่งวางสายนะคะ… เขายังไม่รู้ว่าหนูอยู่ในนี้." ข้างหลังเสียงเธอมีเสียงหยดน้ำ กับอะไรบางอย่างที่ดังทุกสิบวินาที.*

### A3 · «คดีที่ปิดไปแล้ว»
- **Logline:** คุณเป็นนักข่าวสายอาชญากรรม ได้กล่องเอกสารคดีฆาตกรรมเมื่อ 20 ปีก่อนที่ปิดไปแล้ว จากคนส่งนิรนาม — ในกล่องมีของที่พิสูจน์ว่าคนที่ติดคุกไม่ได้ทำ
- **Meter:** **ความน่าเชื่อถือ** (พิมพ์ข่าวโดยไม่เช็ก/กล่าวหาผิด = เครดิตหด; ถึง 0 = ถูกฟ้อง/ไม่มีใครเชื่อ)
- **Tone:** สืบสวนจริงจัง ประเด็นสังคม จริยธรรมสื่อ
- **Branch hook:** ลงข่าวเร็วเพื่อยอด vs ตรวจสอบให้แน่; ปกป้องแหล่งข่าว vs เปิดโปง
- **Endings:** คดีถูกรื้อ ความจริงชนะ (`justice`) / ถูกกว่าเป็นข่าวปลอมทำลายคนบริสุทธิ์ซ้ำ (`ruin`) / รู้ความจริงแต่ระบบใหญ่เกินสู้ (`bury`) / คนส่งกล่องคือฆาตกรตัวจริงที่อยากปิดปากคุณ (`bait`, secret)
- **เปิดเรื่อง:** *กล่องกระดาษวางหน้าประตูออฟฟิศตอนคุณมาถึงเช้าวันจันทร์ ไม่มีชื่อผู้ส่ง มีแค่เลขคดีเขียนด้วยปากกาเมจิก. ข้างในเป็นสำเนาสำนวนเก่า ซองรูปถ่าย และนาฬิกาข้อมือผู้ชายเรือนหนึ่ง ด้านหลังสลักชื่อที่ไม่ใช่ชื่อของคนที่ศาลตัดสินว่าผิด.*

## แนว B — จีบ / โรแมนซ์ (romance)

### B1 · «สายที่โทรผิด»
- **Logline:** เที่ยงคืนคุณโทรหาเพื่อนสนิท แต่กดผิดหนึ่งเลข — ปลายสายเป็นคนแปลกหน้าที่ก็นอนไม่หลับเหมือนกัน แล้วคุยกันยาวจนเช้า
- **Meter:** **ใจเธอ/เขา** (ตอบมุก/จริงใจถูกจังหวะ = ใจขึ้น; ตอแหล/ก้าวร้าว = ใจตก; ถึง 0 = อีกฝ่ายวางสายถาวร)
- **Tone:** อบอุ่น ตลกเบาๆ บทสนทนาเยอะ slice-of-life
- **Branch hook:** จะบอกความจริงว่าโทรผิดไหม; เปิดใจก่อน vs ระวังตัว; นัดเจอ vs รักษาระยะไว้เป็นเสียงในสาย
- **Endings:** เจอกันจริง (`meet`) / เป็นเพื่อนที่โทรหากันทุกคืน (`friends`) / เขามีคนอยู่แล้ว (`bittersweet`) / เป็นเสียงที่ค่อยๆ หายไป (`fade`) / **อีกฝ่ายคือคนที่คุณตั้งใจโทรหาแต่แรก** (`fated`, secret)
- **เปิดเรื่อง:** *"ฮัลโหล" เสียงผู้ชายที่ไม่ใช่เสียงเพื่อนคุณ. คุณกำลังจะขอโทษแล้ววาง แต่เขาพูดต่อก่อน "ถ้าโทรมาด่าเรื่องที่จอดรถ ผมขอบอกก่อนว่าผมไม่มีรถ." คุณหลุดขำ. นาฬิกาบอกตีสิบสอง ยี่สิบสาม และคุณก็ยังไม่ได้วางสาย.*

### B2 · «รับหน้าที่เป็นแฟนปลอม 7 วัน»
- **Logline:** เพื่อนร่วมงานที่คุณแอบไม่ถูกชะตา ขอให้คุณเป็นแฟนปลอมไปงานแต่งพี่สาวเขาที่ต่างจังหวัด แลกกับการที่เขาจะโหวตให้โปรเจกต์คุณผ่าน — เจ็ดวันในบ้านเขา
- **Meter:** **ความสนิท** (เล่นตามบทเนียน + เข้าใจครอบครัวเขา = สนิทขึ้น; หลุดคาแรกเตอร์/ทะเลาะ = ตก; ถึง 0 = แผนแตก โป๊ะทั้งงาน)
- **Tone:** โรแมนซ์คอเมดี้ enemies-to-lovers ครอบครัวไทยต่างจังหวัด
- **Branch hook:** เล่นตามบท vs แอบมีใจจริง; เข้าข้างเขา vs เข้าข้างครอบครัวเขาตอนเกิดดราม่า
- **Endings:** ของปลอมกลายเป็นจริง (`real`) / จบดีลแบบมืออาชีพ ต่างคนต่างไป (`deal`) / ยายเขาจับได้แต่เก็บเป็นความลับให้ (`grandma`) / คุณรู้ว่าเขาแอบชอบคุณมาตั้งแต่แรก แผนนี้คือข้ออ้าง (`setup`, secret)
- **เปิดเรื่อง:** *เขาวางกาแฟลงตรงหน้าคุณ ทั้งที่ไม่เคยซื้อให้ใคร แล้วพูดเสียงเบา "ผมมีข้อเสนอ แต่คุณต้องสัญญาว่าจะไม่หัวเราะ." คุณหัวเราะทันที. เขาถอนหายใจ "งานแต่งพี่สาวผม เสาร์นี้ ที่บ้านผมเชื่อว่าผมมีแฟนแล้ว และผมโกหกมาสามเดือน."*

### B3 · «ร้านซ่อมของที่พังไปแล้ว»
- **Logline:** คุณเปิดร้านเล็กๆ รับซ่อมของที่คนอื่นบอกว่าซ่อมไม่ได้ วันหนึ่งมีคนเอานาฬิกาที่หยุดเดินตั้งแต่วันที่คนรักเขาจากไปมาให้ซ่อม — และเขากลับมาทุกสัปดาห์
- **Meter:** **ความไว้ใจ** (ฟังจริง/ไม่เร่ง = ไว้ใจขึ้น; สอดรู้/ตัดสิน = ตก; ถึง 0 = เขาไม่กลับมาอีก)
- **Tone:** ละมุน เยียวยา healing slow-burn
- **Branch hook:** ซ่อมให้เดินต่อ vs รักษาไว้ให้หยุดตรงเวลานั้น; ถามเรื่องเขา vs ให้พื้นที่
- **Endings:** ต่างเยียวยากันและกัน (`heal`) / เป็นมิตรภาพที่งดงาม (`kind`) / เขาพร้อมเดินต่อและบอกลา (`goodbye`) / นาฬิกาเรือนนั้นเป็นของคุณเองที่ลืมไปแล้ว (`mirror`, secret)
- **เปิดเรื่อง:** *"ร้านคุณเขียนว่าซ่อมได้ทุกอย่างที่คนอื่นซ่อมไม่ได้ใช่ไหม" เขาวางกล่องกำมะหยี่ลงบนโต๊ะ ในนั้นมีนาฬิกาพกเงินเรือนเก่า เข็มค้างอยู่ที่ 6 นาฬิกา 14 นาที. "ผมไม่อยากให้มันเดิน" เขาพูด "ผมแค่อยากรู้ว่าทำไมมันถึงหยุดตรงนั้น."*

## แนว C — เอาชีวิตรอด / ภัยพิบัติ (survival)

### C1 · «ลิฟต์ค้างชั้น 23»
- **Logline:** ไฟดับทั้งตึกตอนสี่ทุ่ม คุณติดอยู่ในลิฟต์กับคนแปลกหน้าสี่คน ออกซิเจนเริ่มน้อย โทรศัพท์ไม่มีสัญญาณ — และมีคนหนึ่งในนั้นโกหกว่าตัวเองเป็นใคร
- **Meter:** **ออกซิเจน/สติกลุ่ม** (ทำให้กลุ่มแตกตื่น = ทรัพยากร/ความร่วมมือหด; ถึง 0 = คนเริ่มตายหรือทำร้ายกัน)
- **Tone:** ระทึกพื้นที่ปิด human drama ความไว้ใจ
- **Branch hook:** นำกลุ่ม vs ตามคนอื่น; งัดประตูเสี่ยงตก vs รอความช่วยเหลือ; แชร์น้ำ/แบต vs เก็บไว้เอง
- **Endings:** ทุกคนรอด (`allout`) / รอดคนเดียวด้วยราคาที่ต้องจ่าย (`alone`) / ช่วยคนที่โกหกจนได้ความจริง (`truth`) / กลุ่มร่วมมือกันจนถึงที่สุด (`together`) / ตึกนี้ไม่มีชั้น 23 ในผัง (`nofloor`, secret)
- **เปิดเรื่อง:** *ลิฟต์กระตุกแล้วหยุด ไฟกะพริบสองที ก่อนจะเหลือแค่แสงสีแดงของปุ่มฉุกเฉิน. คนห้าคนในกล่องเหล็กมองหน้ากัน. ผู้ชายสูทเทากดปุ่มอินเตอร์คอมรัวๆ ไม่มีเสียงตอบ. "ไม่เป็นไร" เขาพูด เสียงมั่นใจเกินสถานการณ์ "ผมทำงานตึกนี้ อีกเดี๋ยวก็มา" — แต่ป้ายชื่อพนักงานที่เขาห้อยคอ เป็นชื่อบริษัทที่ย้ายออกไปตั้งแต่ปีที่แล้ว.*

### C2 · «น้ำมาแล้ว»
- **Logline:** น้ำท่วมฉับพลันกลางดึกในหมู่บ้านริมคลอง คุณต้องพาคนในบ้าน — ยายที่เดินไม่ได้ หมาอีกสองตัว และของสำคัญ — ขึ้นที่สูงก่อนน้ำถึงหลังคา
- **Meter:** **เวลา/ระดับน้ำ** (ตัดสินใจช้า/เลือกผิด = น้ำสูงขึ้น; ถึงเพดาน = ติดในบ้าน)
- **Tone:** ภัยพิบัติจริง อิงประสบการณ์น้ำท่วมไทย เลือกยากเชิงศีลธรรม
- **Branch hook:** เอาของมีค่า vs เอาแต่คน; ช่วยเพื่อนบ้านที่ตะโกนขอ vs รีบพาครอบครัวตัวเอง; ขึ้นหลังคา vs ลุยไปศูนย์อพยพ
- **Endings:** ทุกชีวิตปลอดภัย (`safe`) / รอดแต่สูญเสีย (`loss`) / ช่วยคนอื่นจนกลายเป็นฮีโร่ของซอย (`hero`) / ยึดติดของจนสาย (`greed`) / เช้ามาน้ำลด เหลือแค่คุณจำได้ว่าเลือกอะไร (`reflect`)
- **เปิดเรื่อง:** *คุณตื่นเพราะเท้าแตะน้ำเย็นข้างเตียง. เปิดไฟ — ไฟไม่ติด. แสงจากมือถือส่องเห็นน้ำสีน้ำตาลไหลเข้าใต้ประตูห้อง สูงขึ้นเรื่อยๆ อย่างเงียบเชียบ. ในห้องข้างๆ ยายยังหลับ. หมาเริ่มเห่า. โทรศัพท์เด้งข้อความจากเทศบาล "ประกาศเตือน: ระดับน้ำคลอง…" แล้วสัญญาณก็หายไป.*

### C3 · «เที่ยวคนเดียวแล้วหลง»
- **Logline:** คุณไปเทรกกิ้งคนเดียวบนดอยทางเหนือ พลาดทางตอนหมอกลง มือถือแบตเหลือ 12% ฟ้าใกล้มืด และอุณหภูมิกำลังลดลงเรื่อยๆ
- **Meter:** **อุณหภูมิร่างกาย** (เดินผิดทาง/เปียก/ไม่พัก = ตก; ถึงจุดวิกฤต = hypothermia)
- **Tone:** เอาตัวรอดสายธรรมชาติ ความรู้จริง (survival tips ที่ถูกต้อง)
- **Branch hook:** เดินหาทางต่อ vs อยู่กับที่รอความช่วยเหลือ; ใช้แบตโทร vs เก็บไว้เป็น GPS; ก่อไฟ vs ประหยัดแรง
- **Endings:** ทีมกู้ภัยเจอ (`rescued`) / เดินออกมาได้เอง (`selfout`) / รอดแบบเฉียดตาย ได้บทเรียน (`lesson`) / ทำถูกทุกอย่างตามหลัก แล้วเช้ามาก็รอด (`bythebook`) / เจอกระท่อมร้างที่มีคนอยู่มาก่อน (`cabin`, secret)
- **เปิดเรื่อง:** *หมอกลงเร็วกว่าที่พยากรณ์บอก. เมื่อกี้ยังเห็นธงผ้าผูกทางอยู่เลย ตอนนี้รอบตัวเป็นสีขาวทึบไปหมด. คุณหยิบมือถือขึ้นมา — 12% ไม่มีสัญญาณ เข็มทิศในแอปหมุนวน. ลมพัดผ่านคอเสื้อ เย็นจนขนลุก. ป่ารอบตัวเงียบผิดปกติ เหลือแค่เสียงหายใจของคุณเอง.*

## แนว D — ชีวิตออฟฟิศ / สังคมไทย (satire)

### D1 · «วันแรกที่ THEOS Corp»
- **Logline:** วันแรกของงานในบริษัทสตาร์ทอัพที่ทุกคนพูดภาษา synergy — คุณต้องเอาตัวรอดในวัฒนธรรม toxic positivity, meeting ที่ควรเป็นอีเมล, และเจ้านายที่เรียกทุกคนว่า "แฟมิลี่"
- **Meter:** **แบตใจ (mental battery)** (เข้าประชุมไร้สาระ/โดน gaslight = แบตหด; ตั้ง boundary สำเร็จ = ได้คืน; ถึง 0 = burnout ลาออกกลางเรื่อง)
- **Tone:** เสียดสีองค์กร ตลกร้าย โคตรไทย relatable (คนวัยทำงานกดแชร์แน่)
- **Branch hook:** เล่นตามการเมืองออฟฟิศ vs ตรงไปตรงมา; รับงานเกินหน้าที่เพื่อภาพ vs ปฏิเสธ; แฉในกรุ๊ปไลน์ vs อดทน
- **Endings:** ได้เลื่อนขั้นแต่กลายเป็นสิ่งที่เคยเกลียด (`become`) / ลาออกไปมีชีวิต (`quit`) / เปลี่ยนวัฒนธรรมได้จริงจากข้างใน (`change`) / เอาตัวรอดแบบ NPC เงียบๆ ถึงเกษียณ (`survive`) / **บริษัทคือ pyramid scheme** (`scheme`, secret)
- **เปิดเรื่อง:** *"ยินดีต้อนรับสู่แฟมิลี่นะคะ!" HR ยิ้มกว้างพลางยื่นเสื้อยืดสกรีนคำว่า HUSTLE ให้. "ที่นี่เราไม่มีลำดับขั้นนะ เราแบน เราคือทีมเดียวกัน" — เธอพูดขณะพาคุณเดินผ่านห้องกระจกที่เขียนว่า 'สำหรับผู้บริหารเท่านั้น'. นาฬิกาบอก 9 โมงเช้า. อีเมลในกล่องของคุณมี 47 ฉบับ ทั้งที่เพิ่งได้อีเมลบริษัทเมื่อสิบนาทีก่อน.*

### D2 · «กลุ่มไลน์หมู่บ้าน»
- **Logline:** คุณเป็นนิติบุคคลคนใหม่ของหมู่บ้านจัดสรร ต้องจัดการกลุ่มไลน์ที่มีทั้งป้าที่บ่นทุกเรื่อง ลุงที่ส่งข่าวปลอม และดราม่าที่จอดรถที่บานปลายเป็นสงคราม
- **Meter:** **ความสงบในหมู่บ้าน** (ตอบผิด/เข้าข้าง = ดราม่าลาม; ไกล่เกลี่ยเก่ง = สงบ; ถึง 0 = ประชุมใหญ่ถอดถอนคุณ)
- **Tone:** เสียดสีสังคมไทยยุคกลุ่มไลน์ ตลกจิกกัด
- **Branch hook:** ลบข่าวปลอม vs ปล่อยผ่าน; เข้าข้างเสียงส่วนใหญ่ vs ทำสิ่งที่ถูก; เปิดโปงกรรมการที่โกงส่วนกลาง
- **Endings:** หมู่บ้านสามัคคีจริง (`unity`) / ลาออกไปอยู่คอนโด (`escape`) / จับได้ว่าใครโกงส่วนกลาง (`expose`) / กลายเป็นป้าดราม่าเสียเอง (`become`) / กลุ่มไลน์คือคุณคุยกับบอตทั้งหมู่บ้าน (`bots`, secret)
- **เปิดเรื่อง:** *เที่ยงคืนตรง มือถือคุณเด้งไม่หยุด. กลุ่ม "หมู่บ้านสุขสันต์ 🏡 (ห้ามขายของ)" มีข้อความใหม่ 213 ฉบับ. เริ่มจากป้าสมศรีถ่ายรูปรถที่จอดขวางหน้าบ้าน จบที่ลุงคนหนึ่งส่งลิงก์ข่าวว่าน้ำประปาหมู่บ้านมีสารเคมี. คุณเพิ่งรับตำแหน่งนิติฯ มาได้สามชั่วโมง.*

### D3 · «เดดไลน์ตี 4»
- **Logline:** ฟรีแลนซ์ดีไซเนอร์ต้องส่งงานลูกค้าตอนเช้า แต่ลูกค้าส่งคอมเมนต์ "ทำให้มันปังกว่านี้" ตอนตีหนึ่ง — คืนนี้คุณต้องเลือกว่าจะยอม จะสู้ หรือจะเท
- **Meter:** **สติ+กาแฟ** (แก้ตามใจลูกค้าไม่จบ = หด; ตั้งขอบเขต/คิดเงินเพิ่มสำเร็จ = ได้คืน; ถึง 0 = หลับคาโต๊ะ พลาดส่ง)
- **Tone:** เสียดสีชีวิตฟรีแลนซ์ relatable ตลกเจ็บ
- **Branch hook:** แก้ฟรีไม่จบ vs คิดเงินเพิ่ม; ยอมลูกค้า vs ยืนยันงานดีไซน์; ส่งงานทั้งที่ไม่พอใจ vs ขอเลื่อน
- **Endings:** ได้เงิน + ลูกค้าประจำที่ดี (`win`) / ไฟลุก เลิกรับลูกค้านี้ (`fire`) / งานปังจนได้ลูกค้าใหม่บอกต่อ (`viral`) / ยอมทุกอย่างจนหมดไฟ (`burnout`) / "ลูกค้า" คือตัวคุณเมื่อ 5 ปีก่อน (`self`, secret)
- **เปิดเรื่อง:** *01:03 น. Notification เด้ง. "พี่คะ ดูแล้วน่ารักมากเลย 🥰 แต่รู้สึกว่ามันยังขาดอะไรไปนิดนึง ทำให้มันว้าวกว่านี้ได้ไหมคะ เดี๋ยวเช้าลูกค้าใหญ่ดู 🙏 (งบเท่าเดิมนะคะ)". คุณมองไฟล์ที่แก้มาแล้ว 14 เวอร์ชัน. กาแฟแก้วที่สามเย็นชืด. เคอร์เซอร์กะพริบรอคำตอบ.*

## แนว E — ย้อนยุค / ประวัติศาสตร์ไทย (historical)

### E1 · «จดหมายจากบางกอก พ.ศ. 2475»
- **Logline:** คุณเป็นเสมียนหนุ่มในกรุงเทพฯ ช่วงเปลี่ยนแปลงการปกครอง 2475 เช้าวันที่ 24 มิถุนายน คุณตื่นมาพบว่าถนนเต็มไปด้วยทหาร และเพื่อนสนิทของคุณอยู่คนละฝั่ง
- **Meter:** **ความปลอดภัย** (เลือกข้างผิดที่ผิดเวลา/พูดพลาด = เสี่ยง; ถึง 0 = ถูกจับ)
- **Tone:** ดราม่าประวัติศาสตร์ อิงเหตุการณ์จริง (คณะราษฎร) — เล่นมุมคนธรรมดาที่ติดอยู่กลางประวัติศาสตร์
- **Branch hook:** ช่วยเพื่อนที่อยู่ฝั่งตรงข้าม vs เอาตัวรอด; เชื่อข่าวลือ vs หาความจริง; มีส่วนร่วม vs เป็นผู้สังเกตการณ์
- **Guardrail:** เล่าผ่านสายตาคนธรรมดา ไม่ตัดสินฝ่ายใดฝ่ายหนึ่ง อิงข้อเท็จจริงที่ตรวจสอบได้ ไม่แต่งบุคคลจริงให้พูดสิ่งที่ไม่มีหลักฐาน
- **Endings:** ผ่านเหตุการณ์มาได้และเข้าใจมัน (`witness`) / ช่วยเพื่อนไว้ได้ (`friend`) / เลือกอุดมการณ์จนต้องเสียสละ (`ideal`) / อยู่รอดแต่โลกเปลี่ยนไปตลอดกาล (`newworld`)
- **เปิดเรื่อง:** *เสียงฝีเท้าสม่ำเสมอปลุกคุณตื่นก่อนไก่ขัน. คุณเปิดหน้าต่างห้องเช่าย่านบางลำพู เห็นทหารเดินเป็นแถวมุ่งไปทางลานพระบรมรูปทรงม้า ปืนสะพายไหล่ ใบหน้าเคร่งขรึม. เจ้าของบ้านกระซิบจากบันได "อย่าเพิ่งออกไปไหนนะ… เขาว่ากันว่าคืนนี้บ้านเมืองจะไม่เหมือนเดิม."*

### E2 · «ตลาดน้ำเมื่อร้อยปีก่อน»
- **Logline:** คุณเป็นลูกสาวเจ้าของเรือขายของในตลาดน้ำสมัยรัชกาลที่ 5 พ่อป่วยหนัก คุณต้องพายเรือค้าขายเลี้ยงครอบครัว ในยุคที่ผู้หญิงทำการค้าเองไม่ใช่เรื่องปกติ
- **Meter:** **ทุนรอน** (ตัดสินใจค้าขายผิด/ถูกโกง = ทุนหด; ถึง 0 = ต้องขายเรือ)
- **Tone:** ดราม่าย้อนยุคอบอุ่น เกร็ดวิถีชีวิตริมน้ำ ประเด็นบทบาทหญิง
- **Branch hook:** ขายตัดราคา vs รักษาคุณภาพ; ไว้ใจพ่อค้าคนกลาง vs ค้าเอง; แต่งงานเพื่อฐานะ vs สู้ด้วยตัวเอง
- **Endings:** กิจการรุ่งเรือง (`thrive`) / รักษาเรือของครอบครัวไว้ได้ (`keep`) / กลายเป็นแม่ค้าที่คนทั้งตลาดนับถือ (`respect`) / เสียเรือแต่ได้อิสระ (`free`)
- **เปิดเรื่อง:** *ฟ้ายังไม่สาง สายหมอกลอยเหนือผิวน้ำ. คุณจุดตะเกียงท้ายเรือ จัดของขึ้นให้เรียบร้อย — มะพร้าว กล้วย ขนมใส่ไส้ที่แม่ทำไว้ก่อนนอน. ในห้องท้ายเรือ พ่อไอเสียงแห้ง. "ระวังตัวนะลูก" เสียงพ่อแผ่ว "ตลาดน้ำวันนี้… พ่อไปกับเจ้าไม่ได้แล้ว."*

## แนว F — ไซไฟ / เหนือธรรมชาติ (ไม่ใช่ผี) (speculative)

### F1 · «แอปที่รู้ว่าคุณจะตายเมื่อไหร่»
- **Logline:** แอปใหม่ไวรัลชั่วข้ามคืน กรอกข้อมูลแล้วมันบอกวันที่คุณจะตายได้แม่นยำ — เพื่อนคุณคนหนึ่งเพิ่งได้วันที่เป็นพรุ่งนี้ และมันไม่เคยผิด
- **Meter:** **เวลาที่เหลือ** (นับถอยหลัง; การตัดสินใจบางอย่างขยับวันได้ — พิสูจน์ว่าอนาคตเปลี่ยนได้ไหม)
- **Tone:** ไซไฟจิตวิทยา ตั้งคำถาม free will vs โชคชะตา
- **Branch hook:** เชื่อแอป vs ท้าทายมัน; พยายามช่วยเพื่อน vs ปล่อยให้เป็นไปตามนั้น; หาคนสร้างแอป vs หนี
- **Endings:** พิสูจน์ว่าเปลี่ยนอนาคตได้ (`freewill`) / ทุกอย่างเป็นไปตามที่ทำนาย (`fate`) / แอปทำนายถูกเพราะมันเป็นคนทำให้เกิด (`cause`) / ปิดแอปแล้วเลือกใช้ชีวิตโดยไม่รู้ (`unknow`) / วันที่ของคุณคือวันนี้ (`today`, secret)
- **เปิดเรื่อง:** *เพื่อนส่งสกรีนช็อตมาในแชต หน้าจอแอปสีขาวสะอาด มีแค่ตัวเลขตรงกลาง: "เวลาที่เหลือของคุณ: 19 ชั่วโมง 42 นาที". ข้างล่างมีข้อความเล็กๆ "แม่นยำ 99.998%". คุณพิมพ์ตอบว่า "มุกอะไรวะ 555" แต่เพื่อนไม่ตอบ. สามวินาทีต่อมา แอปเด้งขึ้นบนมือถือของคุณเอง ให้กรอกข้อมูล.*

### F2 · «สายด่วนถึงตัวเองเมื่อวาน»
- **Logline:** โทรศัพท์บ้านเครื่องเก่าที่เพิ่งย้ายเข้ามาดังตอนตีสอง ปลายสายคือเสียงของคุณเอง โทรมาจากเมื่อวาน เพื่อเตือนไม่ให้ทำสิ่งที่กำลังจะทำ
- **Meter:** **ความเป็นเหตุเป็นผล (paradox)** (เปลี่ยนอดีตมากไป = timeline พัง; ถึงขีด = ลบตัวเองออกจากเส้นเวลา)
- **Tone:** ไซไฟ time-loop ปวดหัวแบบสนุก ปริศนาซ้อน
- **Branch hook:** เชื่อคำเตือน vs ทำตามใจ; บอกความจริงกับคนรอบตัว vs เก็บเงียบ; พยายามแก้อดีต vs ยอมรับ
- **Endings:** แก้ไทม์ไลน์สำเร็จ (`fixed`) / ติดในลูปตลอดกาล (`loop`) / เข้าใจว่าทุกสายคือคุณในแต่ละรอบ (`chain`) / วางสายแล้วใช้ชีวิตต่อ (`letgo`) / คุณคือคนที่โทรไปเตือนคนอื่น (`caller`, secret)
- **เปิดเรื่อง:** *โทรศัพท์บ้านสีครีมเครื่องนั้นไม่เคยดังเลยตั้งแต่คุณย้ายเข้ามา จนกระทั่งตีสอง. คุณคิดว่าโทรผิด แต่พอรับ เสียงปลายสายทำให้ขนลุกทั้งตัว — มันคือเสียงของคุณเอง. "ฟังให้ดีนะ อย่าเพิ่งวาง" เสียงนั้นพูดเร็ว "พรุ่งนี้เช้าอย่ารับสายจากเบอร์ที่ขึ้นต้นด้วย 02 เด็ดขาด ไม่งั้นเราจะเป็นแบบนี้."*
